Object Relational Mappers (ORMs)

An object-relational mapper (ORM) is a library that automates the transfer of data stored in relational databases tables into objects that are more commonly used in application code.

I.e they provide a bridge between R.Db, and Python objects. This allows you to work in a single language rather than two separate languages.
